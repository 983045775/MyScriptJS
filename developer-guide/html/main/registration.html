MyScriptJS Developer Guide

Registration

In order to use MyScriptJS, users are required to register. Only requests from valid users will be considered by the HTTP server.

To become a valid user, follow the below procedure:

  1. Create an account on the developer portal and confirm your registration by clicking on the link you received by mail.
  2. Login to your account and click CDK samples on the main page.
  3. Check the box I want a free 90-day developer license for MyScript Cloud Development Kit and click I accept.

You will then receive the following email:

        Dear MyScript developer,

        Thank you for registering on our developer portal.

        You now have a free 90-day access to the MyScript Cloud service, that provides online handwriting recognition, at the following URL: https://cloud.myscript.com/

        Use your email address and the password used for the developer portal to enter the MyScript Cloud website. Your account gives you access to all the features of our recognition web service, but you are limited to 200 requests per day.

        Samples are available for download on the developer portal, just click CDK samples in your account.

        Before using the code samples available for download or writing your own code, you will need to generate your application and HMAC keys.

        1. Login to your account in your MyScript Cloud service dashboard.
        2. Click Create application, then Open to edit this new application.
        3. Click Generate application key and check the Enable HMAC box to create an HMAC key.

        Don't forget to read the documentation. If you ever have questions related to the MyScript Cloud service, please read our FAQ or visit the community forums.

        Happy programming,

        The MyScript team.

Application and HMAC keys

If you are working on code samples, both keys have to be pasted in the 'index.html' file of all code samples. Replace the existing keys (var applicationKey and var hmacKey) and run the code.

If you are working on your own application, then both keys have to be passed to the doSimpleRecognition method of the recognizer.

You may need to pass other parameters depending on how you are invoiced. These parameters allow us to monitor traffic and bill you correctly in accordance with your contract.
