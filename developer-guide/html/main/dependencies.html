MyScriptJS Developer Guide

Dependencies

The following Javascript libraries are required in order to use MyScriptJS:

    • Q is a tool for creating and composing asynchronous promises, and thus managing the asynchronous communication between the server and the final display,
    • CryptoJS contains 4 modules in charge of HMAC key-related issues:
      • core-min.js
      • x64-core-min.js
      • sha512-min.js
      • hmac-min.js

These libraries can be downloaded for free on GitHub. Note that if you use Bower, there is no need to download them, as their latest version will be automatically used.

We strongly recommend that you use Bower, as it will manage all your frameworks, libraries, assets and utilities.

Besides, HandJS (optional) can be useful as it is a polyfill intended for supporting pointer events on every browser.
