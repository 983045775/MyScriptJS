MyScriptJS is a free and open-source JavaScript library providing the easiest way to use MyScript Cloud handwriting recognition in your web app.

Handwriting input on the web? We've got you covered

MyScriptJS provides default implementation for common tasks involved in your handwriting user interface, including ink capture, rendering and requests to MyScript Cloud.

Start your project quickly

MyScriptJS is modular so you can pick only the objects you need. Code samples and an extensive documentation are provided so you can get it up and running easily.

Made for the long run

Using MyScriptJS instead of a custom wrapper for MyScript Cloud makes it easier to keep up with our latest developments. You can also share code or tips with the community, or contribute a new feature!
