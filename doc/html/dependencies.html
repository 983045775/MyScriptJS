MyScript JS Developer Guide

Dependencies

The following libraries are required to use MyScript JS:

    • Q.JS
    • CryptoJS

The Q.JS library allows the recognition to be displayed, regardless of your browser.

The CryptoJS library contains 4 modules in charge of HMAC key-related issues.

These libraries can be downloaded for free on GitHub. Note that if you use Bower, there is no need to download them, as their latest version will be automatically used.

We strongly recommend that you use Bower, as it will manage frameworks, libraries, assets and utilities.
